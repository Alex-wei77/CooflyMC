# CooflyMC-Rquirements

### 1. Mods 文件使用指南

CooflyMC所需文件，更新建议直接从`Releases` 打包下载，直接替换原 `mods` 文件夹。

也可以查看更新日志，手动替换对应版本。



### 2. 加入服务器指南

**Minecraft 版本:** Java 1.17.1

**前置插件:** Fabric

**需配备本 Repo 的 mods 文件夹**
